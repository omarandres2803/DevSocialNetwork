# Developers Social Network

This application is built using the MERN stack:

- MongoDB
- Express
- ReactJS
- NodeJs

---

Some other key concepts / libraries used on this project are :

- JWT (JSON Web Tokens)
- Async / Await
- React Hooks
- Redux
- Postman HTTP Client
- Mongoose
- Bcrypt Password Hashing
- Heroku and git deployment
