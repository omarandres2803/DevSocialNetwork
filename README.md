# Developers Social Network

This application is built using the `MERN` stack:

- MongoDB
- Express
- ReactJS
- NodeJs

---

Some other key `concepts` used on this project are :

- JWT (JSON Web Tokens)
- Async / Await
- React Hooks
- Redux
- Postman HTTP Client
- Mongoose
- Bcrypt Password Hashing
- Heroku and git deployment

---

Used `dependencies` :

- jsonwebtoken
- bcryptjs
- express
- request
- mongoose
- gravatar
- express-validator
- config

---

Used `Dev Dependencies `:

- nodemon
- concurrently
